# 安全基线检查项目

[English](README.en.md)

这是一个用于安全基线检查的辅助项目，面向 Windows、Linux/Unix、数据库服务等常见检查对象。项目围绕安全检查表开展核查，整理检查结果，并生成报告和证据材料。

## 项目简介

项目可以根据检查表内容，对目标系统的账号策略、访问控制、审计配置、服务端口、远程管理、数据库配置等内容进行核查。

它适合用于安全基线检查、课堂作业、复查验收、运维自查和安全加固前后的对比记录。

## 支持对象

- Windows Server 虚拟机
- Linux/Unix 主机
- Docker 中运行的数据库服务
- MySQL/MariaDB 等常见数据库场景

## 输出内容

项目通常会生成：

- Excel 安全检查报告
- 截图证据目录
- 命令输出或检查记录
- 与检查对象对应的结果汇总

输出示例：

```text
centos安全检查报告.xlsx
centos安全检查证据\
```

截图文件会使用简短名称，方便查看和归档，例如：

```text
row05_身份鉴别.png
row06_口令策略.png
row22_服务端口.png
row24_超时锁定.png
```

## 项目特点

- 支持 Windows 图形化界面检查。
- 支持 Linux/Unix SSH 检查。
- 支持数据库服务基础安全检查。
- 支持生成 Excel 报告。
- 支持保存截图证据。
- 支持把长命令输出整理为更容易查看的关键证据。

## 典型场景

- 按安全基线表检查服务器配置。
- 对多台虚拟机生成统一格式报告。
- 为安全课程、实验或作业整理检查材料。
- 对加固前后的系统状态进行对比。
- 为复核、验收或归档准备证据截图。
